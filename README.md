# Azure-AZ104-Mastery

This repository documents my journey of mastering the [AZ-104](https://learn.microsoft.com/en-us/credentials/certifications/azure-administrator/?practice-assessment-type=certification) domains.

## The stack
- Cloud Provider: Microsoft Azure
- Framework: [Azure Well-Architected Framework](https://learn.microsoft.com/en-us/azure/well-architected/)
- Methodology: Incremental Phase-based deployment

## Phases 
- **Phase 1 Governance and Identity**: Enforce regional and cost boundaries via Azure Policy. *in preparation*
- **Phase 2 Network**: Implement a Hub-and-Spoke network with bidirectional peering.*Planned*
- **Phase 3 Storage and connectivity**: Zero-trust data storage that is invisible to the outside world.*Planned*
- **Phase 4 Compute and Monitoring**: A functional, monitored, and backed-up application environment.*Planned*
